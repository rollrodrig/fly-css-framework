Grid columns

The grid numeration start from 1 to 12, example fy-gcs5

fy-gc[1-12] The grid is always active
fy-gcs[1-12] Active the grid only when the screen is > 560px
fy-gcm[1-12] Active the grid only when the screen is > 768px
fy-gcl[1-12] Active the grid only when the screen is > 980px
